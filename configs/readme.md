# Configuration Files

This directory is a placeholder for community contributions.

## How to Contribute
*   **Snort:** Submit custom `.rules` files in a `snort/` subdirectory.
*   **Wazuh:** Submit `ossec.conf` or `agent.conf` examples in a `wazuh/` subdirectory.
*   **pfSense:** Submit sanitized `config.xml` snippets in a `pfsense/` subdirectory.
